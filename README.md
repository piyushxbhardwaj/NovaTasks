# NovaTasks

A production-quality full-stack Task Management application built using React 19 and FastAPI.

## Tech Stack
- **Frontend:** React 19, Vite, Tailwind CSS, React Router, Axios, TanStack Query, React Hook Form, Zod, React Hot Toast
- **Backend:** FastAPI, SQLAlchemy, SQLite, python-jose, passlib (bcrypt), Pydantic
- **Deployment:** Vercel (Frontend), Render (Backend)
